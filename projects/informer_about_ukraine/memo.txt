仮想環境が必要な場合

仮想環境を生成
python -m venv 仮想環境名

仮想環境を有効にする
仮想環境名/scripts/activate

=======================================

requirements.txtにあるライブラリーをインストール
pip install -r requirements.txt

=======================================

サーバーの起動
python manage.py runserver

ブラウザからアクセスするためのurl
localhost:8000

========================================
